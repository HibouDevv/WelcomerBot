Features

WelcomerBot is a powerful yet easy-to-use Discord bot that welcomes new members with style. Here's what it can do:

🎉 Welcome New Members

Automatically sends a welcome message when a user joins your server. You choose the channel and customize the message.

⚙️ Slash Command Setup

Set everything up with intuitive slash commands – no messy configurations.

💬 Custom Welcome Message

Use /setembed to craft a stylish embedded welcome message, or use plain text if you prefer.

🧪 Test Before You Launch

Use /testwelcome to preview the welcome message before members see it.

🖼️ Add Visuals

Use /setimage to include an image or GIF in your welcome message.

✉️ Send Direct Messages

Enable private welcomes using /setdm – great for onboarding or personalized greetings.

♻️ Reset with One Click

Made a mistake in your message? Just use /resetmessage to start fresh.

ℹ️ Learn More

Use /about or /help to learn what each command does, right from Discord.
